# LuckMyBall
### Controls
- Walk : W-A-S-D  
- Jump : Spacebar  
- Shoot Bullets : Left Mouse Button  
- Throw TNT : Right Mouse Button  

### How to play
Scene1
1. Find the red door
2. Find 3 keys in the map
3. The door will disapear

Scene2
1. Find the red door
2. Find 3 keys in the map
3. The door will disapear
4. end game

Notice : the capsule model is the key  
You can  
- Collect Ammo  
- Collect Grenard  
- When you died, rebirth at checkpoint
